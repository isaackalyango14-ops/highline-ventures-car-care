# Image Setup - Save All Images to /public/images/

Copy all your photos into `public/images/` with these exact filenames:

| Filename | Which photo |
|---|---|
| `facility-panorama.jpg` | Wide panoramic exterior (long shot of whole facility) |
| `facility-front.jpg` | Front view with Highline Ventures sign on canopy |
| `facility-close.jpg` | Closer shot of facility exterior |
| `workshop-staff.jpg` | Workshop interior with 3 staff visible + sign overhead |
| `workshop-interior.jpg` | Inside workshop with blue hydraulic lift + air compressor |
| `alignment-bay.jpg` | The wheel alignment pit interior (green floor, 4-post) |
| `tyre-changer.jpg` | Blue IT-9J tyre changer machine |
| `wheel-balancer.jpg` | Red/black JohnBean 300L wheel balancer with tyre mounted |
| `alignment-machine.jpg` | Red/yellow hydraulic alignment tower machine |
| `alignment-pit.jpg` | 4-post alignment pit from the front (cross-bar view) |
| `hydraulic-lift.jpg` | Blue platform hydraulic lift (flat lift in workshop bay) |
| `price-board.jpg` | Price signboard on the orange wall |
| `og-image.jpg` | Any wide photo, 1200x630px for social media sharing |

After placing the images run: `npm run dev` and open http://localhost:3000
